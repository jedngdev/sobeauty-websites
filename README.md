# sobeauty-websites

Monorepo containing all client websites built for SoBeauty beauticians.
Each subfolder is a self-contained website for one client.

## Structure
```
sobeauty-websites/
└── <client-slug>/   # one per client, self-contained
```

## Creating a new client site
Clone one of the base templates, paste into a new folder named after the client slug, then apply the brand brief.
